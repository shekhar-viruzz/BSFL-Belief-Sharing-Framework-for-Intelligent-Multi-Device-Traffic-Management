# BSFL - Belief Sharing Framework for Intelligent Multi-Device Traffic Management

## 1. Application Description

BSFL (Belief Sharing Framework for Intelligent Multi-Device Traffic Management) is an AI-powered decentralized traffic management system that enables vehicles and Roadside Units (RSUs) to collaboratively detect, share, and respond to traffic conditions before congestion occurs.

Traditional traffic systems usually detect congestion after vehicles are already trapped. BSFL introduces a proactive approach where vehicles generate traffic beliefs based on their environment and share this information through Vehicle-to-Vehicle (V2V) and Vehicle-to-Infrastructure (V2I) communication.

The system classifies traffic conditions into:

- Normal Traffic
- Slow Traffic
- Congestion

Vehicles exchange beliefs with nearby vehicles, while RSUs aggregate information from multiple vehicles to build a global traffic understanding and provide dynamic rerouting decisions.

---

# System Features

- AI-based vehicle traffic state prediction
- Decentralized belief sharing between vehicles
- V2V communication framework
- V2I communication with RSUs
- RSU-based congestion analysis
- Dynamic route optimization
- SUMO-based traffic simulation
- Real-time monitoring dashboard
- Multi-device integration
